Discord Wordle Bot

A bot showing Wordle stats of the members in a discord channel.

I'm a sucker for Wordle, so I created a bot that tracks the Wordle stats of the people who play Wordle in our discord channel. It provides a fun way to engage players, and make the game more competitive.

The bot reads the messages, posted by the Wordle game, in the channel and keeps track of:

  • Games played
  • Games lost
  • Average guess amount
  • Amount of games resulting in 1 guess, 2 guesses, etc.
It tracks every players' stats, and provides a leaderboard showing the best players in the channel.

I couldn't find a bullet-proof way to host the bot for free, without having to run it on my machine 24/7, so I ended up with a version that requires me to run it manually. But it does remember the stats and can read up to 500 missed Wordle messages, so the chances of it not being up-to-date is tiny.
