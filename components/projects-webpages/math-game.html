Math Game

A simple math game with a local Leaderboard. Perfect for motivating students to improve their math skills.

I used this project to introduce the following to my skillset:

  • Node.js
  • Vite
  • React
  • TypeScript
  • Tailwind

It was also a perfect opportunity to try out:

  • LocalStorage for the leaderboard
  • framer-motion for animations
  • react-circle-flags for language selection

LocalStorage stores the language selection and the scores in the browser, which means it stays for as long as you don't clean your browser memory, or use the "Reset Progress" button. Might change it to a database if I get to expand it at some point.

The most interesting, and confusing at first, part was the context, which ultimately was an immense help to provide languages in a practical way. It turned out quite well, making it easy to add languages in the future.

Another challenge I had was having the menu at the top being separate from the pages, while having the pages fill the rest of the screen. The solution was, in the end, quite simple. I created a Layout component returning a div wrapping around the menu and a div containing the rest of the page. That way the page had a parent element with a real size.
